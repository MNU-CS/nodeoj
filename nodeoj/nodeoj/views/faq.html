常见问题

问：可以使用哪种编程语言?
答：目前只提供C、C++、Java和Python 2。

问：如何进行读取和输出?
答：只能stdin/stdout（使用标准输入/输出），不允许读取和写入任何文件。在C/C++中可以使用scanf读取，printf输出。

问：各种状态的含义是什么?
答：如下列表：
  • Queuing：代码已经提交，正在等待测评
  • Compiling：代码正在编译
  • Running：编译成功，正在进行测评
  • Acepted：代码正确
  • Wrong Answer：答案错误。程序输出了错误的答案，或者多输出或者少输出了空格，换行等符号
  • Runtime Error：运行时错误。引发的原因有很多,包括但不仅限于使用未初始化的指针,数组越界,堆栈溢出,除数为零
  • Time Limit Exceeded：超出时间限制。可能是代码中包含死循环，也可能是算法不够优化
  • Memory Limit Exceeded：内存超出限制。你使用的内存太多了，可能由于使用了过大的数组，或是忘记释放使用过的内存导致内存泄漏
  • Output Limit Exceeded：你的输出超过标准答案的三倍
  • Compilation Error：编译错误。请检查代码在自己的机器上能否正常编译，以及是否选错了语言。在比赛中编译错误不会增加罚时
  • System Error：系统内部错误，请联系管理员重测

问：各语言的编译选项是怎样的？
答：如下列表：

  • C：gcc source.c -o main -O2 -Wall -DONLINE_JUDGE
  • C++：g++ source.cpp -o main -O2 -Wall -DONLINE_JUDGE
  • Java：javac Main.java
  • Python 2：Python采用直接运行，不进行编译
问：如何编写代码?
答：以A+B Problem为例：

C语言
#include <stdio.h>
int main()
{
    int a, b;
    scanf("%d%d", &a, &b);
    printf("%d\n", a + b);
    return 0;
}

C++
#include <iostream>
int main()
{
    using namespace std;
    int a, b;
    cin >> a >> b;
    cout << a + b << endl;
    return 0;
}

Java
import java.io.BufferedInputStream;
import java.util.Scanner;
public class Main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(new BufferedInputStream(System.in));
        try {
            int a = sc.nextInt(), b = sc.nextInt();
            System.out.println(a + b);
        } finally {
            sc.close();
        }
    }
}

Python 2.7
a, b = map(int, raw_input().split())
print a + b
